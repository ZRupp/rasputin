Rasputin WebGL Visualizer

Based on a detailed DEM raster file, a TIN is constructed. When rasputin_web is run with -a, heights, aspects and slopes are used to compute the location of various avalanche risks as forecasted by varsom.no.

Rasputin is developed by Expert Analytics AS and is open source (GPL-3.0 due to some dependencies). This tool is in early beta stage, and the source code can be found at Github.
